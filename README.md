# Pruebas de Localización — NKS Full Localización 18.0

Validación funcional de la localización venezolana (sin nómina) por el equipo de consultores NIKKOSOFT.

**Página publicada:** https://lstuard-gonzalez.github.io/pruebas-localizacion/

| Recurso | Descripción |
|---|---|
| [Estadísticas por consultor](https://lstuard-gonzalez.github.io/pruebas-localizacion/) | Tablero en vivo con el avance de todo el equipo. |
| [Checklist interactivo](https://lstuard-gonzalez.github.io/pruebas-localizacion/checklist-validacion-localizacion.html) | 19 secciones · 109 verificaciones. Tocas tu nombre y marcas ✓/✗ con notas. |
| [data/](data/) | Un JSON por consultor, escrito automáticamente por el checklist (con historial de commits). |
| [Guía funcional](GUIA-FUNCIONAL-PRUEBAS-LOCALIZACION.md) | Detalle funcional de los módulos bajo prueba. |
| [Manual de validación](MANUAL-VALIDACION-LOCALIZACION.md) | Procedimiento general de la validación. |

## Cómo funciona

1. El consultor (Danny, Chriselys o Luis) abre el **checklist** y toca su nombre.
2. Marca cada prueba (✓ pasó / ✗ falló + nota de lo que vio). **Todo se guarda solo**
   en `data/` de este repositorio — nada que descargar ni subir.
3. El **tablero de estadísticas** muestra en vivo el avance global y por sección de cada
   consultor, con sus fallas reportadas (se refresca cada 2 minutos).
